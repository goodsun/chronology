# Mythos
### ── The myth of the emergence of symbols ──

> 和題：『ミュトス ── 記号創発の神話』 ／ 語り部の口上：「わたしは、記号」

世界史を、**読んで面白い物語**にする試み。
目的は、世界史の「大まかな流れ」を、楽しみながら、忘れられない形で掴むこと。

> タイトル `Mythos` は、語り部「わたし（記号）」の最初の名であり、最後の名。
> 読み終えたとき、それがずっと語り部の名前だったと分かる——タイトルそのものが伏線。

## コンセプト

- **群像劇として描く** — 各地域（西アジア／地中海／インド／中国 …）は、互いを知らずに並走する登場人物。全部を見渡せるのは読者だけ。物語の山場は、別々に張られた伏線が**ある瞬間に交わって回収されるとき**。
- **語り部は「わたし」＝進化する記号** — 人類が「遺したい・伝えたい」と願った営みそのものが一人称で語る。`しるし → 文字 → 神話（ミュートス）→ ロゴス → 技術 → …` と姿を変え、名を増やしていく。
- **経緯・仕組みを語る** — 「何が起きたか」より「**なぜ・どうやってそうなったか**」を主役に。村が国になり、国が帝国になる力学を、その都度ちゃんと見せる。
- **壮大な叙事詩風**。
